# Documentation Index

Use this folder as the primary home for project docs.

## Folders
- `docs/guides/`: setup guides and tutorials
- `docs/operations/`: runbooks, quick command sheets, and triage notes
- `docs/project/`: project-level planning and hardware context
- `docs/handoff/`: dated system state and session handoff records
- `docs/reference/pdfs/`: reference PDFs and external manuals
- `docs/notes/`: working notes (for example shopping or procurement notes)

## Media
- `assets/images/oak/`: camera/media artifacts used by docs

## Start Here
- `docs/guides/GETTING_STARTED_GAUSSIAN_SPLATS.md`
